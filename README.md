# API com Python - FastAPI

Este repositório reúne estudos e práticas sobre **FastAPI**, framework moderno e de alto desempenho para construção de APIs em Python.  
O conteúdo está organizado em capítulos que abordam desde conceitos introdutórios até tópicos avançados como manipulação de banco de dados, autenticação, testes e deploy.

## Estrutura do projeto

📂 API-COM-PYTHON-FASTAPI  
├── 01-introducao_aplicacao_Rest  
├── 02-introducao_fFastApi_RESTful_Assincronas  
├── 03-fastAPI_primeiros_passos  
├── 04-banco_dado_relacional_python_db_api  
├── 06-manipulacao_bd_fastApi_assincrono  
├── 07-autenticacao_autorizacao_fastApi  
├── 08-teste_api_restful_assincrona_fastApi  
├── 09-deploy_api_fastApi_assincrona  
├── 10-api_bancaria_assincrona_fastApi  
└── README.md  

## Objetivo
- Explorar os fundamentos de APIs RESTful  
- Aprender a criar endpoints assíncronos com FastAPI  
- Integrar banco de dados relacionais  
- Implementar autenticação e autorização  
- Realizar testes automatizados  
- Preparar e realizar deploy de aplicações
